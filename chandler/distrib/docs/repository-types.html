Types in the Chandler repository

This document describes the types that you can use when you are working with parcel.xml. These types are provided by the Chandler repository and are the basic building blocks for Chandler Items. You can use them by supplying the name of the type as the value of the itemref attribute on the <type> element used in parcel.xml.

Repository type name Description Parcel.xml Declaration Syntax Parcel.xml Value syntax
Boolean A Python boolean <type itemref="Boolean"> True or False
Class a Python class <type itemref="Class"> The fully qualified name of the class as a string
Complex a Python complex number <type itemref="Complex" /> a Python complex number literal
DateTime a mxDateTime.DateTime <type itemref="DateTime"> An ISO8601 date/time string
DateTimeDelta a mxDateTime.DateTimeDelta <type itemref="DateTimeDelta"> use mxDateTime.DateTimeDelta's DateTimeDeltaFrom constructor
Dictionary A Python dictionary <type itemref="Dictionary"> A string containing key value pairs: "key1:value1, ..., keyn:valuen"
Enumeration one of a number of string values - string1, ..., stringn <Enumeration itsName="fontFamilyEnumType">
<values>DefaultUIFont</values>
<values>SerifFont</values>
<values>SansSerifFont</values>
<values>FixedPitchFont</values>
</Enumeration>

<Attribute itsName="fontFamily">
<type itemref="docSchema:fontFamilyEnumType"/>
<initialValue type="docSchema:fontFamilyEnumType">DefaultUIFont</initialValue>
</Attribute> 		One of the string values defined by the enumeration type
Float A Python float <type itemref="Float"/> a Python floating point literal
Integer A Python int <type itemref="Integer"/> a Python integer literal
List A list of values <type itemref="List"/> A string containing a list of values: "value1, ... , valuen"
Lob A binary object <type itemref="Lob" /> A MIME encoded binary value
Long A Python long <type itemref="Long" /> a Python long literal
RelativeDateTime a mxDateTime.RelativeDateTime <type itemref="RelativeDateTime" /> use mxDateTime.RelativeDateTime's RelativeDateTimeFrom constructor
SingleRef A uni-directional reference to an Item <type itemref="SingleRef"/> The string representation of a UUID, which is the UUID of the item to reference
String A Unicode string <type itemref="String"/> A Python unicode string literal
Struct A collection of named fields <Struct itsName="ColorType" itemClass="osaf.framework.blocks.DocumentTypes.ColorStruct">
<fields key="red"></fields>
<fields key="green"></fields>
<fields key="blue"></fields>
<fields key="alpha"></fields>
<implementationTypes key="python">osaf.framework.blocks.DocumentTypes.ColorType</implementationTypes>
</Struct>

<Attribute itsName="foregroundColor">
<type itemref="docSchema:ColorType"/>
<initialValue type="docSchema:ColorType">0,0,0,255</initialValue>
</Attribute>		 Use Python to assigne values to the appropriate fields of the Struct.
Tuple A tuple of values <type itemref="Tuple"/> A string containing comma separated values: "value1, ... , valuen"
URL A URL - a URL string <type itemref="URL"/> A string containing a valid URL
UUID A UUID <type itemref="UUID"/> The string form of a UUID>
