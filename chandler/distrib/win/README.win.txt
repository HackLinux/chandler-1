Chandler README.win
-------------------

* Running Chandler * 

To run chandler, use the batch file "Chandler.bat" in this directory, like
so:

 Chandler.bat

Note, for this release you must be in the same directory as Chandler.sh and
Chandler.py when running the shell script.  Also, you can double-click the
Chandler.bat file from Windows Explorer.
