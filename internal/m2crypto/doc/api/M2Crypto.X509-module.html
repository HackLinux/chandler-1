Module M2Crypto.X509

M2Crypto wrapper for OpenSSL X509 API.

This module has evolved based on the needs of supporting X.509 certificate operations (mainly attribute getters for authentication purposes) from within an SSL connection.

Open Source Applications Foundation (OSAF) has extended the functionality to make it possible to create certificates programmatically.
