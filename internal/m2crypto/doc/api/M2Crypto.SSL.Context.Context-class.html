load_cert(self, certfile, keyfile=None, callback=<function passphrase_callback at 0x00AA3670>)

Load certificate and private key into the context.

'certfile' - File object containing the PEM-encoded certificate.

'keyfile' - File object containing the PEM-encoded private key. Default value of None indicates that the private key is to be found in 'certfile'.

'callback' - Callable object to be invoked if the private key is passphrase-protected. Default callback provides a simple terminal-style input for the passphrase.
